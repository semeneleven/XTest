Код Грея. Короткі теоретичні відомості

Відображені (рефлексні) коди будуються таким чином, що сусідні кодові комбінації, на відміну від простих двійкових кодів, розрізняються цифрою тільки в одному розряді, тобто кодова відстань між сусідніми кодовими комбінаціями такого коду дорівнює одиниці. З відображених кодів найбільшого поширення набув код Грея.

Перетворення простого двійкового коду в код Грея проводиться за алгоритмом: Yn = Xn; Yi = Xi xor Xi+1 , (xor – підсумовування за модулем 2), де Yi – значення i-го розряду кода Грея; Xi, Xi+1 – відповідні значення розрядів двійкового числа ( i = 1, 2, ..., n, рахуючи, розряд с i = 1 – молодший, i = n – старший ). Тобто, для всіх розрядів, окрім молодшого, значення коду Грея дорівнює сумі за модулем 2 відповідного і наступного більш старшого розрядів двійкового коду. Найстарший розряд зберігається.

Декодування (зворотне перетворення) коду Грея в двійковий код здійснюють за наступною формулою: Xn = Yn; Xi = Xi + 1 xor Yi, де Xn и Yn – значення старшого розряду двійкового коду та коду Грея відповідно (i = n-1, n-2, ..., 1, рахуючи, розряд з i = 1 - молодший, i = n - старший).

  1. Старший розряд переноситься в декодовану послідовність без змін.
  2. Наступні розряди виходять підсумовуванням за модулем 2 відповідно і всіх більш старших розрядів закодованого повідомлення - при 8-ми розрядному повідомленні для декодування 4-го рязрядів необхідно скласти по модулю 2 розряди 4, 5, 6, 7 і 8 кодової комбінації.

Приклади.

  1. Закодувати число 10111101011 кодом Грея.

    Y11 = X11 = 1
    Y10 = X10 xor X11 = 1
    Y9 = X9 xor X10 = 1
    Y8 = X8 xor X9 = 0
    Y7 = X7 xor X8 = 0
    Y6 = X6 xor X7 = 0
    Y5 = X5 xor X6 = 1
    Y4 = X4 xor X5 = 1
    Y3 = X3 xor X4 = 1
    Y2 = X2 xor X3 = 1
    Y1 = X1 xor X2 = 0

    Результат кодування: 11100011110.

  2. Декодувати число 1110101, закодоване кодом Грея.

    X7 = Y7 = 1
    X6 = Y7 xor Y6 = 0
    X5 = Y7 xor Y6 xor Y5 = 1
    X4 = Y7 xor Y6 xor Y5 xor Y4 = 1
    X3 = Y7 xor Y6 xor Y5 xor Y4 xor Y3 = 0
    X2 = Y7 xor Y6 xor Y5 xor Y4 xor Y3 xor Y2 = 0
    X1 = Y7 xor Y6 xor Y5 xor Y4 xor Y3 xor Y2 xor Y2 = 1

    Результат декодування: 1011001.
