Код з перевіркою по модулю q. Короткі теоретичні відомості.

    Принцип побудови даного коду аналогічний з побудови двійкового коду з однією перевіркою на парність по mod 2. Відмінність в побудові полягає в доповненні кодових комбінацій первинного q-ічного коду одним перевірочним розрядом до значення підстави (алфавіту) коду q, тобто якщо кодова комбінація є безліччю з k елементів {a1a2...ak}, де a1, a2, ..., ak – інформаційні елементи кодової комбінації, що приймають значення від 0 до (q-1), то перевірки розряд b1 визначається сумою цих елементів по mod q:

    b1 = q - (a1 + a2 + ... + ak) mod q.

Наприклад:

    Нехай кодова комбінація складається з 4 елементів, тобто k = 4, і алфавіт складається з 4 цифр {0,1,2,3}, т.е. q = 4. Визначимо перевірки розряд і запишемо нову комбінацію. a1 = 1; a2 = 0; a3 = 3; a4 = 2. Перевірки розряд дорівнює b1 = 4 - ( 1 + 0 + 3 + 2 ) mod 4 = 4 - 2 = 2. Комбінація 10322.

Перевірка на наявність помилки:

10322 S = (1 + 0 + 3 + 2 + 2) mod 4 = 0 без помилки;
12322 S = (1 + 2 + 3 + 2 + 2) mod 4 = 2 с ошибкой.
