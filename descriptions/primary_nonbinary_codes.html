Первинні недвійкові коди. Короткі теоретичні відомості.

   Підстави недвійковий кодів завжди більше 2, тобто q >= 3,тому для побудови таких кодів використовують методи теорії комбінаторики: перестановки Pq из q елементів, розміщення Aqmі поєднання (комбінація) Cqз q елементів по m елементів.

    Для кодів, які базуються на перестановках символів алфавіту довжина кодової комбінації n = q = const. Повне число перестановок:

        N = 1 · 2 · 3 ... n = n! = q!.

    Так, наприклад, для коду з алфавітом q = 3 (a, b, c) буде всього 6 перестановок: abc, acb, bac, bca, cab, cba, тобто N = 1 · 2 · 3 = 6.

    Збільшення алфавіту коду призводить до збільшення числа перестановок і відповідно числа кодових комбінацій.

    Відмінною особливістю такого коду є відсутність однакових символів в одній кодової комбінації, тому їх можна віднести до кодів c виявленням одноразових та деяких багаторазових помилок. Дійсно, перетворення будь-якого символу в комбінації призводить до появи в ній неcколько однакових символів, що автоматично виявляються декодером і викличе відмову в прийомі даної комбінації.

    Для кодів, які базуються на розміщеннях символів алфавіту повне число комбінацій:

        N = q · ( q - 1 ) · ( q - 2 ) · ... · ( q - m + 1 ) = q! / ( q - m )!,

де m - число символів алфавіту q, з яких складається кодова комбінація, завжди q > m,а довжина кодової комбінації n = m. Такі коди, на відміну від кодів перестановки, дозволяють виявляти тільки деякі одноразові помилки, коли під впливом перешкод відбувається перетворення, що призводить до подвоєння деяких символів в комбінації. При дворазових помилках і помилках більшої кратності можуть виникнути дозволені комбінації, які не можуть бути виявлені декодером.

    Для кодів, які базуються на певному числі поєднань символів алфавіту, повне число комбінацій:

        N = q! / ( q - m )! · m!,

де m - число символів алфавіту q, з яких складається кодова комбінація, завжди q > m,а довжина кодової комбінації n = m. Такі коди відрізняються від кодів на розміщення відсутністю комбінацій, які відрізняються тільки порядком розміщення символів. Тобто до даних кодів належать кодові комбінації, які відрізняються одна від одної самими символами q. У таких кодах, як і в кодах, які базуються на розміщеннях, в одній комбінації не може бути два однакових символу, такі комбінації легко розпізнаються на приймальній стороні.

    Для кодів на все поєднання в одній комбінації можуть знаходиться різні, в тому числі, і однакові символи. При цьому загальне число комбінацій:

        n = qN,

де m - число символів алфавіту q, з яких складається кодова комбінація, завжди q > m, а довжина кодової комбінації n = m. Для даного коду можна відзначити збільшення кількості кодових комбінацій в порівнянні з кодом на розміщення.

    Змінно-якісний код можна отримати з коду на всі сполучення, якщо накласти на нього деякі обмеження. Так, в комбінації змінно-якісного коду не повинні перебувати поруч однакові символи. Число комбінацій такого коду:

        N = q · (q -1)( n - 1 )

    До перевага коду слід віднести можливість розпізнавання кодових комбінацій і виявлення в них помилок, так як в комбінаціях цього коду два однакових символу не повинні стояти поруч.
