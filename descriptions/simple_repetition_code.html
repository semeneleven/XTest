Код з простим повторенням. Короткі теоретичні відомості.

    В основу побудови даного коду, за аналогією з двійковим, покладений метод повторення вихідної кодової комбінації. Відмінність q-ічного коду від аналогічного довічного полягає в тому, що повторення кодової комбінаяціі q-ічного коду може здійснюватися паралельно в часі введенням додаткової позиційної надмірності. Так, наприклад, при використанні многочастотного коду подвоєння числа частотних позицій забезпечує паралельну передачу кодової комбінації даного слова. При цьому з'являються додаткові переваги перед двійковим кодом: передача інформаційної та перевірочної частин кодової комбінації многочастотного коду проводиться з рознесенням по частоті, що підвищує стійкість коду при селективних завмираннях, характерних для деяких типів бездротових типів зв'язку.

    Алгоритм побудови q-ічного коду з повторенням має вигляд:
    Ai = Bi, i Є [ 1, k ], де
Ai - безліч позицій, призначене для передачі i-го повідомлення;
Bi- безліч позицій, призначене для передачі i-го перевірочного одиничного елемента кодової комбінації;
k - число інформаційних елементів

    Приклад:

при k = 4 комбінація 1032 перетвориться в 10321032.
