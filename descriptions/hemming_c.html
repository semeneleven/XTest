Циклические коды Хэмминга. Краткие теоретические сведения.

    Циклические коды с минимальным кодовым расстоянием dmin = З являются разновидностью кодов Хэмминга. Эти коды способны исправлять одиночные ошибки или обнаруживать все одиночные и двойные ошибки. В отличие от групповых кодов Хэмминга, в циклических кодах проверочные разряды размещаются в конце кодовой комбинации. Длина кодовой комбинации выбирается из условия n = 2r - 1. Образующим полиномом может быть любой неприводимый полином степени r. Разновидностью кодов Хэмминга являются также циклические коды с dmin = 4, которые строятся на основе образующих полиномов для кодов dmin = 3. Циклические коды Хэмминга с dmin = 4 обладают большей корректирующей способностью по сравнению с такими же кодами, но dmin = З и способны обнаруживать ошибки кратности T0 = 3 и менее.

    Образуюший полином циклического кода с dmin = 4 есть произведение двучлена х + 1 на неприводимый полином, который пригоден как образующий для кода с dmin = 3. Длина кодовой комбинации выбирается из условия n = 2r - 1, число проверочных разрядов m = r + 1. Так, если образующий полином имеет вид (х + 1) · (х4 + х + 1) = х5+ х4 + х2 + 1. С помощью этого полинома можно образовать код с числом проверочных разрядов m = r + 1 = 5 длиной n = 2r - 1 = 15. 

    В общем случае многочлены кодов, способных исправлять одиночные, двойные, тройные и т.д. ошибки, можно определить, базируясь на следующем указании Хэмминга:

  1. По заданному n определяем число проверочных разрядов m - n, необходимое для исправления одной ошибки, и строим (n, А)-код.
  2. Рассматривая (n, A)-код как некорректирующий n-разрядный код, определяем n1 количество дополнительных разрядов для обеспечения исправления одной ошибки в этом коде и строим код ( n1, A ).
  3. Повторяя данную процедуру t раз, можно получить код, исправляющий независимые ошибки кратности до t включительно.

    Однако код, построенный таким образом, оказывается неоптимальным с точки зрения количества проверочных символов при заданном n. В этом отношении более совершенен код Боуза - Чоудхури - Хоквингема, который обеспечивает минимальное число проверочных символов при заданном n.

Примечание: после нахождения образующего полинома кодирование и декодирование (исправление ошибок) в циклических кодах Хемминга выполняется аналогично другим циклическим кодам. См., например, теоретические сведения к кодам Абрамсона.
